Workshop Internet of Things

Volg deze handleiding om snel wat leuks met je iota bordje te kunnen ontwikkelen!

Voorbereiding

Installeer de volgende software voordat de workshop begint:

  • Download en installeer de laatste Arduino IDE (nu 1.6.12) voor jouw OS uit deze directory:
    • installs@Google Drive
  • Installeer de driver voor de USB-chip van het ESP8266 bordje (alleen nodig voor Mac en Windows). Deze staan in dezelfde directory als de Arduino IDE. Vergeet niet te rebooten als hier om gevraagd wordt. Sommige Linux systemen hebben issues met rechten op de USB poort, lees hier om dat op te lossen
  • Start de Arduino IDE en ga naar Arduino, Preferences en vul bij Additional Boards Manager URLs dit in: http://arduino.esp8266.com/stable/package_esp8266com_index.json
  • Ga naar Tools, Board: .., Boards Manager en zoek hier op 'esp8266'. Installeer de laatste versie (2.3.0).
  • Selecteer onder Tools, Board, WeMoS D1 R2 & mini
  • Nu heb je een aantal libraries nodig. Download de volgende 4 ZIP-files en voeg deze toe aan de Arduino IDE via Sketch, Include library, Add .ZIP Library.
    • FastLED
    • LM75
    • OneButton
    • pubsubclient
  • Download de repo hier
    of doe een git clone https://github.com/topicusonderwijs/iota.git
  • Ga naar de directory software/arduino/iota-workshop
    en open iota-workshop.ino met de Arduino IDE.

IoTa aansluiten

Als je eenmaal in het bezit bent van een IoTa bordje sluit je deze via USB aan. Selecteer vervolgens onder Tools de volgende opties (vanaf Board):

De waarde voor port zal per OS verschillen.

Vul in de Arduino IDE Editor in de Sketch de WIFI_SSID en WIFI_PASSWORD in.

Vervolgens kun je via Sketch, Upload je code in de ESP8266 laden.

De testsketch verbindt met wifi en toont initieel de waarde van de twee sensoren via de serial monitor. Deze kun je openen via het icoontje rechtsboven in de Arduino IDE. Selecteer in dit nieuwe venster rechtsonder 115200 baud.

Met de knop je kun je het volgende:

  • 1x klikken: led aan/uit
  • dubbelklikken: rgb leds aan/uit (groen)
  • vasthouden: sensorwaarden tonen

Succes!

Toepassingen

  • In de Arduino IDE vind je onder File, Examples meer voorbeelden van de gebruikte libraries. O.a. ESP8266HTTPClient is handig om simpele HTTP requests te doen. Let op: er momenteel issues zijn met requests via https. Gebruik http indien mogelijk.
  • ThingSpeak biedt een platform om allerlei sensorgegevens te verzamelen en in te zien.
  • IFTTT (If This Then That) is een handige clouddienst die verschillende andere clouddiensten (Twitter, Dropbox, Philips Hue etc.) met elkaar kan verbinden. Via het Maker channel kun je hier ook je IoTa bordje mee koppelen. Zie deze sketch voor een voorbeeld.
  • Een aantal voorbeeldsketches vind je in de IoTa repo onder software/arduino/.
  • Mogelijke ideeën om zelf uit te werken:

    Basic:
    • Temperatuurschaal weergeven in rgb (met vinger verhitten)
    • Donker (weinig licht) -> led aan
    Intermediate:
    • Internet enabled button (HTTP/REST/IFTT/MQTT)
    • Bij bepaalde sensor waarde mail sturen via IFTTT.com
    • Publish sensor waarde via MQTT (zichtbaar via HiveMQ)
    • Subscribe via MQTT, status via led of serial (input via client of HiveMQ)
    Advanced:
    • MQTT dashboard (evt. Crouton gebruiken)
    • ESP8266 Webserver
    • Twitter mood ‘ring’ (sentiment analyzer, zie Node-RED demo)
    • Ga los met IFTTT of Node-RED
